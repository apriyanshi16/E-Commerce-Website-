# 🛍️ FashionHub - E-Commerce Website

FashionHub is a modern and responsive **full-stack e-commerce platform** built to provide a seamless online shopping experience. Users can browse products, search items, filter by categories, manage their cart, and securely authenticate to access personalized features.

## ✨ Features

* 🔐 User Authentication (Login & Signup)
* 🛍️ Browse products by category
* 🔍 Search products in real time
* 🎯 Advanced filtering and sorting options
* 🛒 Shopping cart functionality
* ❤️ Wishlist support
* 📱 Fully responsive design
* ⭐ Product ratings and reviews
* 💳 Secure checkout experience
* ⚡ Fast and user-friendly interface

```bash id="rtttx3"

```

```bash id="qmd52s"

```

```bash id="pm4mha"

```

```bash id="z7c2zm"

```

